В шаблонах выполняемых для обычных посетителей сайта определены следующие переменные:
  • $t - глобальная область видимости;
  • $me - переменная содержит необходимые для шаблонизатора данные. В нее входят следующие элементы, все пути идут без завершающего слеша:
    • $me.path - путь для запуска модуля. То есть в модуле guestbook переменная будет содержать http://site.name/site/path/guestbook;
    • $me.www - путь к сайту. То есть http://site.name/site/path;
    • $me.content - веб путь к каталогу контента ;
    • $me.my_content - путь к каталогу контента данного модуля ;
  • $theme - информация о теме.
    • $theme.css - путь к css файлам сайта;
    • $theme.images - путь к картинкам связанным с темой сайта;
В шаблонах административной панели: Для отображения списка используется класс "cms_list", внутри него находятся TBODY с data-type="object", у этих объектов должен быть атрибут data-object-id="object_id()" из которого можно восстановить сам объект.
