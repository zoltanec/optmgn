Виды полей: clist - пользователь выбирает только одно свойство из этого поля, индифидуально для каждого товара multi - к данному полю прикрепляется несколько товаров, в качестве данных: cat:х,y,z - категории из которых необходимо выбрать сопутствующие товары. Например для картошки мы выбираем наполнители из категории x; prod:x,y,z - идентификаторы товаров которые могут быть прикреплены к данному товару.
